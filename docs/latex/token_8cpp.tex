Global keyword-\/to-\/token-\/type mapping used during lexical analysis.

This map is used outside the \doxylink{class_token}{Token} class, typically during tokenization, to look up whether a given word (like "{}\+Geralt"{}, "{}loots"{}, "{}potion"{}) is a keyword and what \doxylink{tokenizer_8h_aa520fbf142ba1e7e659590c07da31921}{Token\+Type} it should be classified as.
