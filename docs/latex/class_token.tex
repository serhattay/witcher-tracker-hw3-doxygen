\doxysubsection{Detailed Description}
\texorpdfstring{$<$}{<} Required for \doxylink{tokenizer_8h_aa520fbf142ba1e7e659590c07da31921}{Token\+Type} definition 

Represents a single lexical unit (token) with its type and content.

Each token is identified during the tokenization phase and consists of\+:
\begin{DoxyItemize}
\item the raw content (e.\+g., "{}\+Geralt"{}, "{}potion"{}, "{}3"{})
\item its type (as defined by the \doxylink{tokenizer_8h_aa520fbf142ba1e7e659590c07da31921}{Token\+Type} enum) 
\end{DoxyItemize}
